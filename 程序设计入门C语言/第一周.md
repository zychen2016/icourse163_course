# 为什么是C语言?
* 现代的编程语言在语法上差异很小
* 几乎都是C-like语言
* 语言的能力/适用的领域主要是由库和传统所决定的

#编译-->运行
* C需要被编译才能运行，所以需要编辑器或者编译器或者IDE(集成开发环境)
